# 連携させる

!!! Info "わんコメについて"

    * 対応しているわんコメのバージョンは v5です。

## 事前準備

* わんコメとOBSを起動します。

## 初期設定

* 連携を選びます。

![Image title](images/link_p01.png)

* OBS と接続します。

![Image title](images/import_p02.png)

* わんコメと接続します

![Image title](images/link_p02.png)

## 設定が終わった後

* 一度設定がおわれば、次からは画面上のボタンを押すだけで接続が完了します。

![Alt text](images/import_use_p01.png)
